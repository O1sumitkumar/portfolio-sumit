# Product Overview

This is a personal portfolio website for Sumit Kumar Jha, a software engineer. The portfolio showcases professional projects and skills in a visually appealing manner.

## Key Features

- Interactive 3D elements using Three.js
- Responsive design for all device sizes
- Dark/light theme toggle
- Project showcase with detailed case studies
- Contact form using AWS SES for email delivery
- Component library with Storybook integration

## Brand Guidelines

- The site uses a minimalist design aesthetic
- Primary brand elements include the custom logo and 3D displacement sphere
- Typography uses Gotham Book and Gotham Medium fonts
- Color scheme is based on a dark/light theme toggle system
